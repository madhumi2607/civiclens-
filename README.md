# CivicLens
**AI-Powered Civic Infrastructure Intelligence for Indian Cities**

🔗 **Live Demo:** https://civiclens-95o9.onrender.com

## What is CivicLens?
CivicLens is a two-sided AI platform that transforms how Indian cities detect, prioritize, and resolve civic infrastructure failures. It connects citizens who report problems with municipal workers who fix them through an AI intelligence layer.

## Features
- **Citizen Reporting** — Zero-download PWA, photo capture + GPS, AI classification in 3 seconds
- **DBSCAN Spatial Clustering** — Groups nearby reports within 500m radius into actionable clusters
- **Root Cause Detection** — Flags recurring infrastructure failures with structural intervention recommendations
- **Auto-Assignment** — Matches work orders to field teams by specialty, zone, and availability
- **Municipal Dashboard** — Live map, priority-ranked action briefs, ward health scores, SLA tracking
- **Analytics** — Resolution trends, team performance, city-wide distribution
- **Multi-City Support** — Works across any Indian city, not hardcoded to one location

## Tech Stack
- **Frontend:** React + Vite, Tailwind CSS, Leaflet.js, Recharts
- **Backend:** Node.js + Express
- **Database:** Supabase PostgreSQL + PostGIS (demo mode uses in-memory store)
- **AI:** Claude Vision API (mock in demo), DBSCAN clustering
- **Deployment:** Render
